# CV Application

A browser-based CV application built with **React** for creating and previewing a professional résumé.

## Features

- Add and edit **General Information** (name, email, phone number)
- Add and edit multiple **Education** entries
- Add and edit multiple **Work Experience** entries
- **Dynamic dropdowns** for each section for better UX
- Live **résumé preview** updating in real-time as forms are filled
- Accessible design with **keyboard navigation** and **ARIA labels**
- Responsive layout that works on **desktop and mobile screens**
- Form validation using **HTML5 required fields**

## Technologies Used

- **React** (functional components, hooks: `useState`, `useEffect`)
- **JavaScript (ES6 Modules)** for state management and form handling
- **CSS3** for styling forms and résumé preview, including Flexbox layouts
- **Vite** for fast development and bundling
